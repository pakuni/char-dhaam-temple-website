# Char Dhaam Temple Website

A beautiful, responsive website dedicated to Char Dhaam Temple in Sitapur, Uttar Pradesh, India.

## About

This website provides information about the Char Dhaam Temple, a significant religious pilgrimage destination in Sitapur district. The site includes details about the temple's location, visiting information, and spiritual significance.

## Sections

1. **Navigation Bar** - Easy navigation to all sections
2. **Hero Section** - Welcoming introduction with call-to-action
3. **About** - Information about the temple's significance
4. **Details** - Key information cards about the temple
5. **Visit** - Practical information for pilgrims
6. **Contact** - Links and contact information
7. **Footer** - Copyright and source attribution

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Navigate through the website using the menu

## Automatic Deployment

This project uses GitHub Actions to automatically deploy to GitHub Pages. Every push to the `main` branch triggers a deployment.

The website will be available at: `https://pakuni.github.io/char-dhaam-temple-website/`
